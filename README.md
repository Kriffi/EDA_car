# Used Cars Price Prediction

## Описание проекта
Данный проект посвящен предварительной обработке данных и разведочному анализу (EDA) набора данных подержанных автомобилей в США. Основная цель работы — очистка данных от шума, заполнение пропусков и подготовка структурированного датасета для последующего обучения модели классификации ценовых категорий.

## Цели и задачи
* **Data Cleaning**: Выявление и обработка пропущенных значений в признаках `year`, `fuel`, `model` и `title_status`.
* **Типизация**: Приведение данных к оптимальным типам (`int`, `category`) для ускорения обработки и экономии памяти.
* **Обработка аномалий**: Удаление статистических выбросов методом межквартильного размаха (IQR).
* **Feature Engineering**: Подготовка базы для генерации новых признаков на следующем этапе.

## Стек технологий
* **Python**
* **Pandas**: Обработка и фильтрация табличных данных.
* **NumPy**: Математические вычисления и работа с пропусками.
* **Jupyter Notebook**: Проведение интерактивного анализа.

## Основные этапы реализации
1. **Заполнение пропусков**:
   - `year`: использована медиана, чтобы избежать влияния экстремальных значений.
   - `fuel` и `title_status`: заполнено модой (самым частым значением).
   - `model`: пропущенные значения заменены на метку `other`.
2. **Оптимизация памяти**:
   - Колонки с текстовыми категориями переведены в тип `category`.
   - Год выпуска преобразован в целочисленный тип `int`.
3. **Фильтрация выбросов**:
   - Рассчитаны границы IQR для года выпуска.
   - Из датасета удалены записи, выходящие за рамки статистической нормы (слишком старые авто или ошибки ввода).

## Структура репозитория
* `practice_cars.ipynb` — ноутбук с пошаговым кодом обработки.
* `vehicles_dataset.csv` — исходный набор данных.
* `vehicles_cleaned.csv` — результат обработки (очищенный датасет).

## Как запустить
1. Склонируйте репозиторий.
2. Установите необходимые библиотеки: `pip install pandas numpy`.
3. Откройте `practice_cars.ipynb` и запустите все ячейки.
